# material-components-harmonyos
A Material Components for HarmonyOS. 一个鸿蒙的 Material 组件库

重要：该项目旨在构建一个 HarmonyOS 的 Material Design 风格的组件，项目目前处于起步阶段，欢迎大家参与贡献。
